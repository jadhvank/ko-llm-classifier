# 한국어 AI 텍스트 판별 데모

한국어 글이 사람이 쓴 것인지 LLM이 생성한 것인지 **브라우저 안에서만** 판별하는 정적 웹 데모입니다. 서버·API·GPU가 필요 없고, 붙여넣은 글은 어디로도 전송되지 않습니다.

**데모:** https://jadhvank.github.io/ko-ai-detector/ <!-- Pages 활성화 후 -->

## 어떻게 동작하나

- 문장 단위로 분해 → **char/word n-gram TF-IDF + 선형 SVM 앙상블**로 각 문장을 채점 → 의심 문장 글자 비율(`ai_ratio`)로 문서 판정(`human` / `maybe` / `ai`).
- 추론이 전부 JavaScript로 구현돼 있어(`app.js`) 정적 호스팅만으로 돌아갑니다. 모델 가중치는 `model.json`(gzip 약 1.4MB)에서 로드됩니다.
- 페이지 로드 시 내장된 참조 벡터로 자체 검증을 수행해, 원 모델과 결과가 어긋나면 경고 배지를 띄웁니다.

## 한계 (반드시 읽어주세요)

- **연구·교육용 데모**입니다. 판정은 통계적 추정이며 절대적 근거가 아닙니다.
- 특정 코퍼스(한국어 청원·뉴스·백과)와 특정 생성 모델로 학습돼, 다른 장르·문체에서는 오탐이 있을 수 있습니다.
- **개인·학술적 불이익을 판단하는 근거로 사용하지 마세요.**
