개인정보처리방침 — 한국어 AI 글 판별기 (신호등)

시행일: 2026-07-21

이 확장 프로그램(“서비스”)은 사용자가 보고 있는 웹페이지의 한국어 본문이 AI로 작성됐을 가능성을 추정해 표시합니다. 아래는 서비스가 데이터를 어떻게 다루는지에 대한 설명입니다.

한 줄 요약: 모든 분석은 사용자의 브라우저 안에서만 이뤄집니다. 페이지 내용, 주소, 분석 결과를 외부로 전송하거나 저장하지 않습니다.

서비스가 접근하는 정보

  • 웹페이지 본문 텍스트: 사용자가 툴바 아이콘을 눌러 분석을 요청하거나, 사용자가 자동 분석 대상으로 직접 등록한 도메인의 페이지가 열릴 때 해당 페이지의 본문 텍스트를 읽습니다. 드래그로 텍스트를 선택한 경우 그 선택 영역을 읽습니다.
  • 현재 탭의 주소(URL): 방문한 도메인이 자동 분석 대상인지 판단하고 사이드 패널에 표시하기 위해 사용합니다.

정보를 어떻게 처리하나요

  • 본문 추출(Mozilla Readability)과 AI 작성 여부 추정을 포함한 모든 계산은 사용자의 기기 안에서만 수행됩니다.
  • 판별 모델과 코드는 확장 프로그램 안에 포함되어 있으며, 실행을 위해 외부 서버에 접속하지 않습니다.
  • 페이지 내용, 주소, 분석 결과를 외부 서버로 전송하지 않고, 원격에 저장하지 않으며, 제3자와 공유하거나 판매하지 않습니다.

저장되는 정보

  • 설정값: 사용자가 등록한 자동 분석 도메인 목록 등 설정은 브라우저 저장소(chrome.storage.sync)에 저장됩니다. 사용자의 브라우저 및 (동기화를 켠 경우) 사용자의 Google 계정 안에만 존재하며, 서비스 제공자가 접근하지 않습니다.
  • 분석 결과: 각 탭의 분석 결과는 현재 브라우저 세션 동안만 임시로 보관(chrome.storage.session)되며, 브라우저를 닫으면 사라집니다.

수집하지 않는 정보

이름, 이메일, 위치, 결제 정보 등 개인 식별 정보를 수집하지 않습니다. 트래킹, 광고, 쿠키, 외부 애널리틱스를 사용하지 않습니다.

유의사항

이 서비스의 판별 결과는 확정이 아니라 통계적 추정입니다. 중요한 판단의 단독 근거로 사용하지 마세요.

변경
